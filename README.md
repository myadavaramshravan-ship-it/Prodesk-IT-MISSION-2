# Prodesk-IT-MISSION-2
# 📌 Project Description
#Hi there! This project is a math logical based site called Cash Flow. This site for our daily expenses from our salary. The deducted or expenses amount should be noted and calculate total expenses of our daily note. The total expenses will be removed from your salary and show remaining balance in our pocket.

## 🚀 Features
- Once we set the salary then it will constant whenever we have to change the salary we can change it.
- The total expenses will be calculated from starting onwards.
- We can remove the expenses from expenses list
- Is there one pie chart for analysis purpose it labled as remaining and salary
- The expense list shows with the names of expenses
## Used Technologies
- HTML(hypertext markup language)
- CSS(Cascaded style sheet)
- javascript
